# PhoenixJob Application

Build a Restful  API for a  Job Portal  application using Node.js, Express and MYSQL.

## Steps to Setup

1. Install dependencies

```bash
npm install
```

2. Run Server

```bash
npm start
```

You can browse the apis at <http://localhost:4000>
